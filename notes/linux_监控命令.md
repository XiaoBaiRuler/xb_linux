# 监控命令

## 1. 进程管理

### 1.1. ps

```mathematica
    ps 默认只显示运行在当前终端中属于当前用户的进程
        -e 显示所有进程
        -f 显示完整格式的输出
        -l 显示长列表
            S: 进程状态(O: 正在运行; S: 休眠; R: 可运行, 正等待运行; Z: 僵化, 已终止但找不到父进程, T: 停止)
    常用命令:
        ps -efl 完整格式打印所有进程信息
```

## 1.2. top

```mathematica
    top 实时显示进程信息
        f 对输出进行排序
        d 修改轮询间隔
        q 退出
```
